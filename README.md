# Heroku

.git config add
```
[remote "heroku"]
    url = https://git.heroku.com/jacob-node-socket.git
    fetch = +refs/heads/*:refs/remotes/heroku/*
```
Procfile  
`web: node index.js`

Deploy using Heroku Git  
`$ heroku login`  
`$ git push heroku master`
